Type Definitions

User

Schéma de l'utilisateur pour la base de données MongoDB. Ce schéma définit les champs requis pour un utilisateur, y compris le nom d'utilisateur, le mot de passe et l'email.
Type:
  • Object
Properties:
Name Type Description
username string Le nom d'utilisateur de l'utilisateur (doit être unique).
password string Le mot de passe de l'utilisateur (doit avoir au moins 6 caractères).
email string L'email de l'utilisateur (doit être unique).
Source:
  • users.js, line 4

UserModel

Modèle Mongoose représentant un utilisateur. Ce modèle permet de créer, lire, mettre à jour et supprimer des utilisateurs dans la base de données MongoDB.
